Steps to upgrade competencies/competency_relationships:

Database:

Create a backup of exiting data by dumping data from competency_type table and competency_relationship table:

> mysqldump tusk competency_type > competency_type.sql
> mysqldump tusk competency_relationship > competency_relationship.sql

Run included sql scripts new_tables.sql, modify_tables.sql and academicLeveltables.sql

mysql tusk < new_tables.sql
mysql tusk < modify_tables.sql
mysql tusk < academicLevelTables.sql 

Data migration:

Run the included perl scripts to migrate data from older tables;

> perl add_version.pl
> perl course_objective_migration.pl
